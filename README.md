# 附件批量重命名插件

一款飞书多维表格边栏插件，用于批量重命名多维表格中附件字段的文件。

## ✨ 功能特性

- **替换模式**: 舍弃原文件名，使用全新的命名规则
- **追加模式**: 保留原文件名，在特定位置添加字符
- **变量支持**: 支持自动序号和自定义字段值
- **实时预览**: 配置时实时预览新旧文件名对比
- **差异高亮**: 新文件名变化部分视觉高亮显示

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 部署到 GitHub Pages

```bash
npm run deploy
```

- React + TypeScript
- Vite
- @lark-base-open/js-sdk
- Semi Design

## 📝 使用说明

1. 在飞书多维表格中打开边栏插件
2. 选择要处理的附件字段和作用范围
3. 配置重命名规则（替换或追加模式）
4. 预览新文件名
5. 点击"开始重命名"执行操作

## 🔧 插件配置

在飞书多维表格边栏插件配置中使用以下 GitHub Pages 地址：

```
https://[your-username].github.io/attachment-rename-plugin/
```
